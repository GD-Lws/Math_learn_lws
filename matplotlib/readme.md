## 图形参数
* linestyle = '-' 线样式
* color = 'red' 颜色
* linewidth = 5.0 线宽
* marker = '*' 点样式
* plt.xlabel()给x轴加标签；plt.ylabel()给y轴加标签
* plt.ylim()x轴的刻度，
* plt.legend()图例
